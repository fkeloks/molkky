# Mölkky

Une application pour jouer plus simplement au célèbre Mölkky !  
> <a href="https://fr.wikipedia.org/wiki/M%C3%B6lkky" target="_blank">Le Mölkky sur Wikipedia</a>

## Installation :

### Développement :

```
npm install;
npm run dev;
```

### Production :


```
npm install;
npm run prod;
```
